# Números, datas e horários
Nível: A1

## Objetivo

Neste tópico, você vai aprender a usar números de 0 a 1000, horários, datas simples, versões de software e expressões básicas de fuso horário em inglês. O foco é entender e produzir frases curtas para reuniões, sprints e mensagens profissionais.

Ao final, você deve conseguir dizer coisas como: `The stand-up is at 9`, `The sprint starts on Monday`, `Version 1.2 is ready` e `I am in BRT`.

## Explicação

### Números de 0 a 1000

Os números mais importantes para o trabalho aparecem em horários, tickets, versões, salas, datas e estimativas.

| Número | Inglês |
|---|---|
| 0 | zero |
| 1 | one |
| 2 | two |
| 3 | three |
| 4 | four |
| 5 | five |
| 6 | six |
| 7 | seven |
| 8 | eight |
| 9 | nine |
| 10 | ten |
| 11 | eleven |
| 12 | twelve |
| 13 | thirteen |
| 20 | twenty |
| 30 | thirty |
| 40 | forty |
| 50 | fifty |
| 100 | one hundred |
| 1000 | one thousand |

De 21 a 99, use hífen na escrita: `twenty-one`, `thirty-five`, `ninety-nine`.

Para números como 105 e 250, use formas simples no A1: `one hundred five`, `two hundred fifty`. Você também pode ouvir `one hundred and five`, especialmente em inglês britânico.

### Horários de reunião

Para dizer a hora de uma reunião, use `at`.

- `at 9` = às 9
- `at 9:30` = às 9:30
- `at noon` = ao meio-dia
- `at midnight` = à meia-noite

Em contexto profissional, é comum escrever a hora de forma direta: `The meeting is at 10:15`.

### Dias, meses e datas

Use `on` para dias da semana e datas específicas.

- `on Monday` = na segunda-feira
- `on Friday` = na sexta-feira
- `on June 10` = em 10 de junho

Use `in` para meses, anos e períodos maiores.

- `in June` = em junho
- `in 2026` = em 2026
- `in the morning` = de manhã

Use `at` para horários exatos.

- `at 9` = às 9
- `at 2:30` = às 2:30

Resumo prático: `at` para hora, `on` para dia/data, `in` para mês/ano/período.

### Datas de sprint

Para falar de sprint, você pode usar frases simples:

- `The sprint starts on Monday.` = A sprint começa na segunda-feira.
- `The sprint ends on Friday.` = A sprint termina na sexta-feira.
- `The review is on June 20.` = A review é em 20 de junho.

Em inglês, meses e dias da semana começam com letra maiúscula: `Monday`, `June`, `Friday`.

### Versões simples

Para versões de software, você pode ler cada parte como número separado:

- `version 1.0` = version one point zero
- `version 1.2` = version one point two
- `version 2.10` = version two point ten

Em frases curtas:

- `Version 1.2 is ready.` = A versão 1.2 está pronta.
- `We need version 2.0.` = Precisamos da versão 2.0.

### Fuso horário básico

Para fuso horário, use frases simples:

- `I am in BRT.` = Estou no BRT.
- `The meeting is at 9 BRT.` = A reunião é às 9 BRT.
- `What time is it in UTC?` = Que horas são em UTC?

Em times internacionais, sempre confirme o fuso quando a hora pode causar confusão.

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| The stand-up is at 9. | A stand-up é às 9. | Use `at` para horário. |
| The planning is at 10:30. | O planejamento é às 10:30. | Horário direto e profissional. |
| The sprint starts on Monday. | A sprint começa na segunda-feira. | Use `on` para dia da semana. |
| The demo is on June 12. | A demo é em 12 de junho. | Use `on` para data específica. |
| The release is in July. | O lançamento é em julho. | Use `in` para mês. |
| Version 1.1 is stable. | A versão 1.1 está estável. | Leia como `one point one`. |
| We have twelve tickets. | Temos doze tickets. | Número + substantivo plural. |
| The call is at 8 BRT. | A chamada é às 8 BRT. | Inclui fuso horário. |

## Erros comuns

1. Não use `in Monday`. O correto é `on Monday`.
2. Não use `on June` quando falar apenas do mês. O correto é `in June`.
3. Não use `on 9` para horário. O correto é `at 9`.
4. Lembre que `forty` não tem `u`. A forma `fourty` está errada.
5. Dias e meses em inglês começam com maiúscula: `Monday`, `Friday`, `June`.
6. Para versões, `point` significa ponto: `1.2` = `one point two`.

## Prática guiada

Complete mentalmente as frases:

- A stand-up é às 9. -> `The stand-up is ___ 9.`
- A sprint começa na segunda. -> `The sprint starts ___ Monday.`
- O release é em junho. -> `The release is ___ June.`
- A versão 2.0 está pronta. -> `Version 2.0 is ___.`

Respostas: `at`, `on`, `in`, `ready`.

## Resumo

Use `at` para horários, `on` para dias e datas específicas, e `in` para meses, anos e períodos. Aprenda números básicos para falar de tickets, horários e versões. Em contexto profissional, frases simples e claras são suficientes: `The meeting is at 9`, `The sprint starts on Monday`, `The release is in June`.
