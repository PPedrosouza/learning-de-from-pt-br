# Gabarito: Números, datas e horários
Nível: A1

## Gabarito dos exercícios

| Exercício | Resposta | Explicação |
|---|---|---|
| 1.1 | forty | `Forty` é a grafia correta de 40; não existe `fourty`. |
| 1.2 | The stand-up is at 9. | Usamos `at` para horários exatos. |
| 1.3 | in June | Usamos `in` para meses quando não há data específica. |
| 1.4 | one point two | Em versões, o ponto é lido como `point`. |
| 2.1 | at | `10:30` é horário exato, então usamos `at`. |
| 2.2 | on | `Monday` é dia da semana, então usamos `on`. |
| 2.3 | in | `July` é mês, então usamos `in`. |
| 2.4 | on | `June 10` é data específica, então usamos `on`. |
| 2.5 | at | `noon` funciona como horário específico. |
| 2.6 | in | Para anos, como `2026`, usamos `in`. |
| 3.1 | twelve | `12` em inglês é `twelve`. |
| 3.2 | twenty | `20` em inglês é `twenty`. |
| 3.3 | thirty-five | De 21 a 99, a escrita composta usa hífen. |
| 3.4 | one hundred | `100` é `one hundred`. |
| 3.5 | one thousand | `1000` é `one thousand`. |
| 4.1 | The stand-up is at 9. | A frase usa `at` para indicar o horário da stand-up. |
| 4.2 | The sprint ends on Friday. | Dias da semana usam `on`. |
| 4.3 | The release is in June. | Meses isolados usam `in`. |
| 4.4 | Version 2.0 is ready. | `Version` introduz a versão, e `ready` significa pronta. |
| 4.5 | I am in BRT. | Para indicar o fuso em que a pessoa está, use `in`. |
| 5.1 | The planning is on Monday. | `Monday` pede `on`. |
| 5.2 | The sync is at 8:15. | Horário exato pede `at`. |
| 5.3 | The release is in August. | Mês isolado pede `in`. |
| 5.4 | The review is on June 22. | Data específica pede `on`. |
| 6.1 | The meeting is at 9. | `on 9` está errado para horário; o correto é `at 9`. |
| 6.2 | The sprint starts on Monday. | Dia da semana usa `on`, não `in`. |
| 6.3 | The release is in July. | Mês isolado usa `in`, não `on`. |
| 6.4 | Version 1.0 is one point zero. | Em inglês, o ponto de versão é `point`, não `comma`. |
| 6.5 | We need forty minutes. | A grafia correta de 40 é `forty`. |
| 7.1 | My stand-up is at 9 BRT. | A resposta informa horário e fuso com `at 9 BRT`. |
| 7.2 | The sprint starts on Monday. | `Starts` combina com `sprint`, e `on` marca o dia. |
| 7.3 | The release is in June. | `in June` é a forma correta para mês. |
