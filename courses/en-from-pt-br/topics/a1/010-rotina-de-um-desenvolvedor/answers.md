# Respostas: Rotina de um desenvolvedor

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | reunião diária curta | `Daily stand-up` é uma reunião curta de alinhamento, comum em equipes de software. |
| 1.2 | code review | `Code review` significa revisão de código. |
| 1.3 | never | `Never` significa nunca. |
| 2.1 | usually | A posição natural é antes do verbo principal: `I usually start...`. |
| 2.2 | always | `Always` significa sempre e vem antes de `have`. |
| 2.3 | sometimes | `Sometimes` significa às vezes e vem antes de `fixes`. |
| 2.4 | never | `Never` deixa a frase negativa sem usar `not`. |
| 3.1 | I often review code. | `Often` vem antes do verbo principal `review`. |
| 3.2 | We usually have lunch at noon. | `Have lunch` é a forma natural para almoçar em inglês. |
| 3.3 | She always checks the tasks. | Com `she`, o verbo recebe `-s`: `checks`. |
| 4.1 | She joins the daily stand-up. | Com `she`, usamos `joins`, não `join`. |
| 4.2 | We often review code on Wednesday. | O advérbio de frequência vem antes de `review`. |
| 4.3 | He starts work at eight. | Com `he`, usamos `starts`. |
| 5.1 | On Monday, I usually check my tasks. | Resposta modelo; qualquer frase correta com `usually` e segunda-feira é aceitável. |
| 5.2 | I always have a lunch break. | Resposta modelo; a frase deve mencionar a pausa para almoço e usar `always` ou `never` corretamente. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | I always write code in the morning. | `Always` significa sempre, e `write code` significa escrever código. |
| 2 | joins | Com `she`, o verbo `join` recebe `-s`: `joins`. |
| 3 | focus time | `Focus time` significa tempo de foco. |
| 4 | sometimes | `Sometimes` significa às vezes. |
| 5 | They often have a code review. | `Often` vem antes de `have`; `code review` é revisão de código. |
| 6 | I never skip my lunch break. | `Never` deve ficar antes do verbo principal `skip`. |
| 7 | We usually plan the week on Friday. | A ordem natural é sujeito, advérbio, verbo e complemento. |
| 8 | On Wednesday, I often fix bugs. | Resposta modelo; a frase deve falar de quarta-feira/trabalho e usar `often` corretamente. |

## Respostas da história

| Pergunta | Resposta | Observação |
|---|---|---|
| 1 | Lucas is a junior developer. | A primeira frase apresenta o trabalho de Lucas. |
| 2 | He usually starts work at nine. | A história diz isso na linha sobre Monday. |
| 3 | He has a code review with Ana. | A atividade acontece depois do almoço na terça-feira. |
| 4 | No, he never skips his lunch break. | `Never` indica que ele não pula a pausa para almoço. |
