# Rotina de um desenvolvedor

Nível: A1

## Objetivo

Neste tópico, você vai descrever uma semana simples de trabalho de uma pessoa desenvolvedora. O foco é falar sobre rotina profissional com frases curtas, usando atividades como reunião diária, revisão de código, pausa para almoço e tempo de foco.

Você também vai praticar advérbios de frequência em inglês: `always`, `usually`, `often`, `sometimes` e `never`.

## Explicação

Em inglês, uma frase simples sobre rotina costuma seguir esta ordem:

`pessoa + advérbio de frequência + verbo + complemento`

Exemplos:

- `I always check my tasks.`
- `She usually joins the daily stand-up.`
- `We sometimes have a code review.`

Os advérbios de frequência indicam com que frequência algo acontece:

| Inglês | Tradução | Observação |
|---|---|---|
| always | sempre | Acontece 100% ou quase 100% das vezes. |
| usually | geralmente | Acontece na maioria das vezes. |
| often | frequentemente | Acontece muitas vezes. |
| sometimes | às vezes | Acontece algumas vezes. |
| never | nunca | Não acontece. |

Na rotina de trabalho, muitas frases usam verbos simples no presente:

- `start` = começar
- `join` = participar de uma reunião ou chamada
- `check` = verificar
- `review` = revisar
- `fix` = corrigir
- `write` = escrever
- `have` = ter ou fazer, dependendo do contexto

Com `he`, `she` ou o nome de uma pessoa no singular, o verbo normalmente recebe `-s`:

- `I review code.`
- `She reviews code.`
- `He starts work at nine.`

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| I usually start work at nine. | Eu geralmente começo o trabalho às nove. | Rotina simples com horário. |
| We always have a daily stand-up in the morning. | Nós sempre temos uma reunião diária de manhã. | `stand-up` é uma reunião curta de alinhamento. |
| She often reviews code after lunch. | Ela frequentemente revisa código depois do almoço. | Com `she`, usamos `reviews`. |
| I sometimes fix bugs in the afternoon. | Eu às vezes corrijo bugs à tarde. | `bug` é um erro no software. |
| He never skips lunch. | Ele nunca pula o almoço. | `never` deixa a frase negativa sem usar `not`. |
| On Friday, we usually plan the next week. | Na sexta-feira, nós geralmente planejamos a próxima semana. | Boa frase para descrever uma semana. |

## Erros comuns

1. Colocar o advérbio no lugar errado em frases simples.

Forma natural: `I usually write code in the morning.`

Evite: `I write usually code in the morning.`

2. Esquecer o `-s` com `he`, `she` ou nome singular.

Forma correta: `Ana usually joins the stand-up.`

Evite: `Ana usually join the stand-up.`

3. Usar `never` com `not` na mesma frase.

Forma correta: `I never skip the code review.`

Evite: `I do not never skip the code review.`

4. Traduzir `daily stand-up` palavra por palavra.

`Daily stand-up` significa uma reunião diária curta, comum em equipes de tecnologia. Não precisa significar literalmente que todos ficam em pé.

## Prática guiada

Complete mentalmente as frases com um advérbio de frequência:

- Eu sempre verifico minhas tarefas: `I ___ check my tasks.`
- Ela geralmente participa da reunião diária: `She ___ joins the daily stand-up.`
- Nós às vezes revisamos código antes do almoço: `We ___ review code before lunch.`
- Eu nunca pulo o almoço: `I ___ skip lunch.`

Agora monte uma semana simples:

- `On Monday, I usually check my tasks.`
- `On Tuesday, I often write code.`
- `On Wednesday, I sometimes review code.`
- `On Thursday, I have focus time.`
- `On Friday, I plan the next week.`

## Resumo

Para falar da rotina de uma pessoa desenvolvedora em inglês A1, use frases curtas no presente. Coloque o advérbio de frequência antes do verbo principal: `I usually write code`. Use vocabulário profissional simples, como `daily stand-up`, `code review`, `lunch break` e `focus time`.
