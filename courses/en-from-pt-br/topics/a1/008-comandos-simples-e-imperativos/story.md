# História: Comandos simples e imperativos

Nível: A1

## História em inglês

Lia is at work. She has a new task.

Her manager says, "Open the app, please."

Lia opens the app.

Then he says, "Click the blue button."

Lia clicks the button.

He says, "Type your name and save the file."

Lia types her name and saves the file.

Now the manager says, "Run the test and check the report."

Lia runs the test. She checks the report.

The report is good.

Her manager says, "Create a folder and send the file to me, please."

Lia creates a folder. She sends the file.

Then he says, "Do not delete the old file."

Lia smiles and says, "OK. I will not delete it."

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Lia is at work. She has a new task. | Lia está no trabalho. Ela tem uma nova tarefa. |
| Her manager says, "Open the app, please." | O gerente dela diz: "Abra o aplicativo, por favor." |
| Lia opens the app. | Lia abre o aplicativo. |
| Then he says, "Click the blue button." | Então ele diz: "Clique no botão azul." |
| Lia clicks the button. | Lia clica no botão. |
| He says, "Type your name and save the file." | Ele diz: "Digite seu nome e salve o arquivo." |
| Lia types her name and saves the file. | Lia digita o nome dela e salva o arquivo. |
| Now the manager says, "Run the test and check the report." | Agora o gerente diz: "Execute o teste e verifique o relatório." |
| Lia runs the test. She checks the report. | Lia executa o teste. Ela verifica o relatório. |
| The report is good. | O relatório está bom. |
| Her manager says, "Create a folder and send the file to me, please." | O gerente dela diz: "Crie uma pasta e envie o arquivo para mim, por favor." |
| Lia creates a folder. She sends the file. | Lia cria uma pasta. Ela envia o arquivo. |
| Then he says, "Do not delete the old file." | Então ele diz: "Não delete o arquivo antigo." |
| Lia smiles and says, "OK. I will not delete it." | Lia sorri e diz: "OK. Eu não vou deletá-lo." |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| `task` | tarefa | Algo que você precisa fazer. |
| `manager` | gerente | Pessoa que coordena o trabalho. |
| `blue button` | botão azul | Cor + objeto: `blue` vem antes de `button`. |
| `save the file` | salvar o arquivo | Comando comum em ferramentas. |
| `run the test` | executar o teste | Expressão muito comum em software. |
| `check the report` | verificar o relatório | Conferir se está correto. |
| `send the file` | enviar o arquivo | Enviar para outra pessoa. |
| `do not delete` | não delete | Comando negativo. |

## Perguntas de compreensão

1. Onde Lia está?
2. O que o gerente pede para Lia abrir?
3. Qual botão Lia deve clicar?
4. O que Lia deve fazer depois de executar o teste?
5. O gerente pede para deletar o arquivo antigo?
