# Respostas: Cumprimentos e apresentações

Nível: A1

## Gabarito dos exercícios

| Item | Resposta | Explicação |
|---|---|---|
| 1.1 | Good morning, everyone. | `Good morning` é o cumprimento adequado pela manhã e `everyone` serve para falar com o grupo. |
| 1.2 | I am a developer. | Em inglês, usamos `a` antes de uma profissão no singular. |
| 1.3 | Nice to meet you. | Essa é a frase padrão para `Prazer em conhecer você`. |
| 1.4 | Hi everyone. | `Hi` é natural em chat e `everyone` se dirige ao grupo. |
| 2.1 | is | A estrutura correta é `My name is...`. |
| 2.2 | a | Profissões no singular normalmente precisam de artigo: `a software engineer`. |
| 2.3 | from | `I am from Brazil` indica país ou origem. |
| 2.4 | at | Para empresa, a estrutura estudada é `I work at...`. |
| 3.1 | Hello, I am Marcos. | `Hello` é `olá`, e `I am` apresenta o nome. |
| 3.2 | My name is Aline. | `My name is...` traduz diretamente `Meu nome é...`. |
| 3.3 | I live in Curitiba. | Para cidade onde mora, use `I live in...`. |
| 3.4 | I work at DevHouse. | Para empresa, use `work at`. |
| 4.1 | I am a developer. | A frase corrige a falta do artigo `a` antes da profissão. |
| 4.2 | My name is João. | A transformação mantém o sentido e troca para a estrutura com `My name is`. |
| 4.3 | I work at CodeNow. | A preposição praticada para empresa é `at`. |
| 5.1 | Hello, my name is Ana. I am a software engineer. | Resposta-modelo válida porque inclui cumprimento, nome e função. |
| 5.2 | Hi everyone, I am Pedro. Nice to meet you. | Resposta-modelo válida para chat, com cumprimento ao grupo e frase de apresentação. |
| 5.3 | Hi, can you hear me? | Essa é a pergunta prática para verificar áudio em chamada. |

## Gabarito do teste

| Item | Resposta | Explicação |

| 1 | Good morning, team. | Às 9h, `Good morning` é apropriado, e `team` se dirige ao grupo de trabalho. |

| 6 | in | Para cidade onde mora, use `live in`. |
| 7 | I am from Brazil. | `from Brazil` indica que a pessoa é do Brasil. |
