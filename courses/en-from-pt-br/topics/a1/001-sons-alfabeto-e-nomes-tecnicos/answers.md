# Gabarito: Sons, alfabeto e nomes técnicos

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
| --- | --- | --- |
| 1 | Can you spell that? | Essa é a pergunta direta para pedir soletração. |
| 2 | dot | Em e-mails e arquivos, **dot** representa o ponto. |
| 3 | at | Em e-mails, **at** representa o arroba. |
| 4 | spell | A frase pede o verbo “soletrar”: **spell your name**. |
| 5 | at | Antes do domínio do e-mail, use **at**. |
| 6 | dot | Em **app.js**, o ponto é dito como **dot**. |
| 7 | dash | Em **qa-team**, o hífen é dito como **dash**. |
| 8 | ana at example dot com | Ao falar e-mail, use **at** para arroba e **dot** para ponto. |
| 9 | main dot P-Y | O ponto é **dot** e a extensão pode ser soletrada letra por letra. |
| 10 | dev dash team | O hífen em nome técnico é dito como **dash**. |
| 11 | Can you repeat that? | **Repeat** significa repetir; a frase é um pedido educado. |
| 12 | My username is br dash coder. | **Username** é nome de usuário e **dash** representa o hífen. |
| 13 | The file name is index dot H-T-M-L. | **File name** é nome de arquivo; a extensão é soletrada. |
| 14 | Resposta pessoal. Exemplo: It is spelled M-A-R-I-A. | A resposta deve usar letras separadas e a estrutura **It is spelled...**. |
| 15 | Resposta pessoal. Exemplo: My GitHub username is dev dash maria. | A resposta deve informar um usuário e usar **dash** para o hífen. |

## Gabarito do teste

| ID | Resposta | Explicação |
| --- | --- | --- |
| 1 | Can you spell that? | É a frase correta para pedir que alguém soletre. |
| 2 | dot | O ponto em **beta.app** é dito como **dot**. |
| 3 | U-R-L | A sigla URL pode ser dita letra por letra: **U-R-L**. |
| 4 | dash | **Dash** representa hífen em nomes como usuários e arquivos. |
| 5 | at | Em e-mails, **at** aparece antes do domínio. |
| 6 | repeat | **Can you repeat that, please?** pede repetição de forma educada. |
| 7 | My email is joao at work dot com. | Use **at** para arroba e **dot** para ponto. |
| 8 | server dot L-O-G | Para nome de arquivo, diga **dot** e soletre a extensão. |
| 9 | A-P-I | Em contexto técnico inicial, diga a sigla letra por letra. |
| 10 | Resposta possível: The file name is readme dot M-D. | A frase deve informar um nome de arquivo usando **The file name is...**. |
