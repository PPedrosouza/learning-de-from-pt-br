# Vocabulário essencial de software
Nível: A1

## Objetivo

Ao final deste tópico, você deve reconhecer e usar palavras muito comuns em ambientes de software: nomes de objetos digitais, partes de sistemas e ações básicas de acesso. O foco é leitura e comunicação simples no trabalho, especialmente em mensagens curtas, telas de sistema e conversas com colegas.

## Explicação

Muitas palavras de tecnologia em inglês aparecem quase iguais em português, como `software`, `app`, `server` e `database`. Isso ajuda, mas também pode gerar confiança demais. Em A1, o mais importante é associar cada palavra a uma situação concreta.

Palavras centrais:

- `code`: código de programação.
- `app`: aplicativo.
- `website`: site.
- `server`: servidor.
- `database`: banco de dados.
- `user`: usuário.
- `password`: senha.
- `login`: acesso, entrada no sistema ou dados de acesso.
- `file`: arquivo.
- `folder`: pasta.
- `error`: erro.
- `test`: teste.

Em frases simples, esses substantivos geralmente aparecem depois de `a`, `the`, `my` ou `your`.

- `a file`: um arquivo.
- `the server`: o servidor.
- `my password`: minha senha.
- `your login`: seu login / seus dados de acesso.

Também é comum usar `open`, `save`, `check`, `fix` e `test` com esse vocabulário.

- `Open the file.`: Abra o arquivo.
- `Save the code.`: Salve o código.
- `Check the server.`: Verifique o servidor.
- `Fix the error.`: Corrija o erro.
- `Test the app.`: Teste o aplicativo.

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| The app is open. | O aplicativo está aberto. | `app` é muito comum em contexto profissional. |
| I have a file. | Eu tenho um arquivo. | `file` é arquivo digital, não fila. |
| The folder is empty. | A pasta está vazia. | `folder` é pasta digital ou física. |
| The server is down. | O servidor está fora do ar. | Expressão comum em TI. |
| This is my password. | Esta é minha senha. | Em situação real, não compartilhe senhas. |
| The login is correct. | O login está correto. | Pode ser o acesso ou os dados usados para entrar. |
| There is an error. | Há um erro. | `an` aparece antes do som vocálico de `error`. |
| We test the website. | Nós testamos o site. | `test` pode ser verbo ou substantivo. |

## Erros comuns

- Confundir `file` com “fila”. `File` é arquivo. “Fila” em inglês é `line` ou `queue`, dependendo do contexto.
- Traduzir `password` como “palavra-passe” no Brasil. Em português brasileiro, use “senha”.
- Pensar que `login` sempre é verbo. Em inglês cotidiano de tecnologia, `login` pode ser substantivo: `my login`. Como verbo, é comum ver `log in`: `I log in`.
- Usar `archive` para arquivo comum. Em tecnologia do dia a dia, `file` é a palavra básica. `Archive` costuma indicar um arquivo compactado ou histórico.
- Confundir `user` com `client`. `User` usa um sistema; `client` pode ser cliente comercial ou cliente técnico.

## Prática guiada

Leia as frases e observe o substantivo principal.

1. `Open the folder.`
2. `Check the database.`
3. `The user has a password.`
4. `This website has an error.`
5. `Test the code today.`

Agora troque a palavra central mantendo a estrutura:

- `Open the file.` -> `Open the folder.`
- `Check the server.` -> `Check the database.`
- `Test the app.` -> `Test the website.`

## Resumo

Este vocabulário aparece em telas, tickets, documentação e conversas de equipe. Para A1, priorize reconhecimento rápido, pronúncia aproximada e frases curtas como `Open the file`, `Check the server` e `Fix the error`.
