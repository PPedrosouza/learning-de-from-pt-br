# História: Vocabulário essencial de software
Nível: A1

## História em inglês

Ana is a new user.
She opens the app.
The login is correct.
Her password is private.
Ana opens a folder.
She saves a file.
The website has an error.
Ana checks the server.
The database is ready.
She tests the code.
The app is okay.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Ana is a new user. | Ana é uma usuária nova. |
| She opens the app. | Ela abre o aplicativo. |
| The login is correct. | O login está correto. |
| Her password is private. | A senha dela é privada. |
| Ana opens a folder. | Ana abre uma pasta. |
| She saves a file. | Ela salva um arquivo. |
| The website has an error. | O site tem um erro. |
| Ana checks the server. | Ana verifica o servidor. |
| The database is ready. | O banco de dados está pronto. |
| She tests the code. | Ela testa o código. |
| The app is okay. | O aplicativo está ok. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| user | usuário; usuária | Pessoa que usa um sistema. |
| app | aplicativo | Palavra curta e frequente. |
| login | login; acesso | Pode ser o acesso ao sistema. |
| password | senha | Informação privada. |
| folder | pasta | Onde arquivos podem ficar. |
| file | arquivo | Não significa fila. |
| website | site | Página ou conjunto de páginas na web. |
| error | erro | Problema em sistema ou mensagem. |
| server | servidor | Máquina ou serviço que responde ao sistema. |
| database | banco de dados | Lugar lógico onde dados são guardados. |
| code | código | Texto de programação. |

## Perguntas de compreensão

1. Ana é uma usuária nova?
2. O login está correto?
3. O que Ana salva?
4. O que tem um erro?
5. O banco de dados está pronto?
