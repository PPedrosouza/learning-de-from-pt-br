# Gabarito: Vocabulário essencial de software
Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | arquivo | Em contexto digital, `file` significa arquivo; não significa fila. |
| 1.2 | senha | `Password` é a palavra usada para senha de acesso. |
| 1.3 | banco de dados | `Database` é o local lógico onde dados são armazenados. |
| 2.1 | file | A frase pede a palavra em inglês para arquivo. |
| 2.2 | server | `Server` é servidor, comum em mensagens de infraestrutura. |
| 2.3 | user | `User` significa usuário ou usuária. |
| 2.4 | code | `Code` é código, especialmente código de programação. |
| 3.1 | The folder is empty. | `Folder` traduz pasta e `is empty` traduz está vazia. |
| 3.2 | The app is open. | `App` é aplicativo e `open` pode indicar aberto. |
| 3.3 | There is an error. | Usa-se `an` antes de `error` por começar com som vocálico. |
| 4.1 | Open the folder. | A estrutura continua igual; só troca `file` por `folder`. |
| 4.2 | Test the website. | `Website` substitui `app` como objeto do verbo `test`. |
| 4.3 | Check the database. | `Database` substitui `server` depois de `check`. |
| 5.1 | Save the file. | É uma instrução curta e natural para salvar um arquivo. |
| 5.2 | The login is correct. | A frase usa `login` como substantivo e `correct` para correto. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | O servidor está lento. | `Server` significa servidor e `slow` significa lento. |
| 2 | folder | Em contexto digital, `folder` é pasta. |
| 3 | password | `Private` combina semanticamente com senha, pois senha deve ser privada. |
| 4 | Check the database. | `Check` é verificar e `database` é banco de dados. |
| 5 | website | A palavra pedida para site é `website`. |
| 6 | The user is new. | `User` é usuário e `is new` significa é novo. |
| 7 | File significa arquivo, não fila. | Este é um falso cognato comum para brasileiros. |
| 8 | I test the code. | A frase usa `test` como verbo e `code` como código. |
