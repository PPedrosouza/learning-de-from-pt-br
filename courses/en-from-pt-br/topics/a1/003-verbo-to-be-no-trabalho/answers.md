# Gabaritos: Verbo to be no trabalho

Nível: A1

## Gabarito dos exercícios

| Exercício | Resposta | Explicação |
|---|---|---|
| 1.1 | **am** | Com **I**, a forma correta do verbo **to be** é **am**. |
| 1.2 | **are** | Com **you**, use **are** no presente. |
| 1.3 | **is** | Com **she**, use **is**. |
| 1.4 | **Are you a developer?** | Em perguntas, **are** vem antes de **you**. |
| 2.1 | **is** | **He** combina com **is**. |
| 2.2 | **are** | **They** combina com **are**. |
| 2.3 | **am** | **I** combina com **am**. |
| 2.4 | **are** | **We** combina com **are**. |
| 3.1 | **I'm a software engineer.** | **I'm** é a contração de **I am**. |
| 3.2 | **You're on the product team.** | **You're** é a contração de **you are**. |
| 3.3 | **He's a backend developer.** | **He's** é a contração de **he is**. |
| 3.4 | **She's in the office.** | **She's** é a contração de **she is**. |
| 4.1 | **I am a developer.** | A frase usa **I am** para dizer cargo. |
| 4.2 | **You are on the product team.** | Para falar de equipe, use **on the team**. |
| 4.3 | **She is a QA engineer.** | **She is** indica que ela é engenheira de QA. |
| 4.4 | **He is in a meeting.** | Para localização/situação, **he is in...** funciona como "ele está em...". |
| 5.1 | **Are you a designer?** | A pergunta com **you are** vira **Are you...?**. |
| 5.2 | **Is she on the backend team?** | A pergunta com **she is** vira **Is she...?**. |
| 5.3 | **Yes, I am.** | Resposta modelo; **No, I'm not.** também seria possível se a resposta fosse negativa. |
| 5.4 | **I am a developer.** | Resposta modelo; outros cargos também servem se a frase começar com **I am** e estiver correta. |

## Gabarito do teste

| Questão | Resposta | Explicação |
|---|---|---|
| 1 | **am** | Com **I**, use **am**. |
| 2 | **is** | Com **he**, use **is**. |
| 3 | **Is she a QA engineer?** | Em perguntas, **is** vem antes de **she**. |
| 4 | **You're** | **You're** é a contração correta de **you are**. |
| 5 | **is** | **Maria** equivale a **she**, então usa **is**. |
| 6 | **are** | Com **we**, use **are**. |
| 7 | **She's remote today.** | **She's** contrai **she is**. |
| 8 | **Is he a designer?** | A pergunta com **he is** vira **Is he...?**. |
| 9 | **Are you a product manager?** | Pergunta com **you are** exige a ordem **Are you...?**. |
| 10 | **She is on the support team.** | Para falar de time, use **on the support team**. |
| 11 | **No, I'm not.** | Resposta curta negativa para **Are you...?** usa **I'm not**. |
| 12 | **They are in the office.** | Resposta modelo; qualquer frase correta com **They are** e localização profissional pode ser aceita. |

## Respostas da história

| Pergunta | Resposta | Observação |
|---|---|---|
| 1 | Ana is a developer. | A história diz **I am a developer**. |
| 2 | Leo is in the office. | A frase é **He's in the office today**. |
| 3 | Maya is a QA engineer. | A história não diz que Maya é desenvolvedora. |
| 4 | Are you a developer? | Leo usa **Are you...?** para perguntar sobre Ana. |
| 5 | Is Maya a QA engineer? | Ana usa **Is Maya...?** para perguntar sobre Maya. |
