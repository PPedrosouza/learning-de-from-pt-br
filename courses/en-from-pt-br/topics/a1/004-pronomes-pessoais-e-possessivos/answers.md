# Respostas: Pronomes pessoais e possessivos

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | She | Ana é uma pessoa feminina, então usamos `she`. |
| 1.2 | It | Repository é uma coisa, então usamos `it`. |
| 1.3 | They | Leo e Max são duas pessoas, então usamos `they`. |
| 1.4 | We | Bia and I inclui quem fala, então usamos `we`. |
| 2.1 | my | A frase começa com `I have`, então a posse é `my`. |
| 2.2 | her | Carla é pessoa feminina, então usamos `her` antes de `ticket`. |
| 2.3 | their | The team pode ser retomado como grupo; `their tool` indica a ferramenta deles/delas. |
| 3.1 | He is a tester. | Marcos é uma pessoa masculina, então vira `he`. |
| 3.2 | It is ready. | App é coisa/sistema, então vira `it`. |
| 3.3 | They are colleagues. | Rafa e Nina são duas pessoas, então vira `they`. |
| 3.4 | We are in the sprint. | You and I inclui eu e você, então vira `we`. |
| 4.1 | She is my colleague. | `She` traduz ela; `my` traduz minha/meu. |
| 4.2 | It is our repository. | Use `it` para repositório e `our` para nosso. |
| 4.3 | They are on their project. | `They` é sujeito; `their` indica posse. |
| 4.4 | You are on your laptop. | `you` é você; `your` é seu/sua. |
| 5.1 | pronome pessoal | `I` funciona como sujeito. |
| 5.2 | possessivo | `our` vem antes de substantivo para indicar nosso/nossa. |
| 5.3 | pronome pessoal | `they` funciona como sujeito plural. |
| 5.4 | possessivo | `his` vem antes de substantivo para indicar dele. |
| 5.5 | pronome pessoal | `it` funciona como sujeito para coisa, sistema ou bug. |
| 6.1 | It is open. | Para bug, usamos `it` como sujeito. |
| 6.2 | Our team is small. | `our` indica nosso/nossa antes de `team`. |
| 6.3 | She is in the meeting. | Para ela, usamos `she`. |
