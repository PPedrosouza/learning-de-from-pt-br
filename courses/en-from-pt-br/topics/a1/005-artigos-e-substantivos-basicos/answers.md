# Gabarito: Artigos e substantivos básicos
Nível: A1

## Gabarito dos exercícios

| Exercício | Resposta | Explicação |
|---|---|---|
| 1.1 | A | `laptop` é singular e começa com som de consoante, então usamos `a`. |
| 1.2 | an | `email` começa com som de vogal, então usamos `an`. |
| 1.3 | The | A frase fala de uma tela específica, então `the` é adequado. |
| 1.4 | a | `keyboard` é singular e começa com som de consoante. |
| 2.1 | a | `ticket` é singular com som inicial de consoante. |
| 2.2 | an | `issue` começa com som de vogal. |
| 2.3 | The | A reunião é específica no contexto da frase. |
| 2.4 | an | `office` começa com som de vogal em `office laptop`. |
| 2.5 | a | `branch` começa com som de consoante. |
| 2.6 | The | A frase fala de um teclado específico. |
| 3.1 | laptops | Plural regular: adiciona `-s`. |
| 3.2 | screens | Plural regular: adiciona `-s`. |
| 3.3 | tickets | Plural regular: adiciona `-s`. |
| 3.4 | branches | Depois de `-ch`, o plural comum recebe `-es`. |
| 3.5 | companies | `company` troca `y` por `ies` no plural. |
| 3.6 | people | `people` é o plural irregular de `person`. |
| 4.1 | I have a laptop. | `a laptop` é a combinação correta; `an laptop` e `a laptops` estão erradas. |
| 4.2 | This is an email. | `email` pede `an` por começar com som de vogal. |
| 4.3 | The branches are new. | `branches` é o plural correto de `branch`. |
| 4.4 | Two people are in the meeting. | Com número plural, usamos `people`, não `a people`. |
| 5.1 | The keyboard is new. | `the` indica o teclado específico da frase. |
| 5.2 | We have two tickets. | Com `two`, o substantivo fica no plural. |
| 5.3 | This is a branch. | `branch` é singular e começa com som de consoante. |
| 5.4 | The meeting is today. | A frase fala de uma reunião definida. |
| 5.5 | Four people are online. | `people` é plural irregular de `person`. |
| 5.6 | I have an email. | `email` usa `an` no singular. |
| 6.1 | laptops | Plural regular de `laptop`. |
| 6.2 | meetings | Plural regular de `meeting`. |
| 6.3 | women | Plural irregular de `woman`. |
| 6.4 | men | Plural irregular de `man`. |
| 6.5 | children | Plural irregular de `child`. |
| 7.1 | I have a laptop. | `laptop` pede `a`, não `an`. |
| 7.2 | We have tickets. | `a` não combina com substantivo plural. |
| 7.3 | This is an email. | `email` começa com som de vogal. |
| 7.4 | The branches are ready. | O plural correto de `branch` é `branches`. |
| 7.5 | Two people are in the call. | Em uso geral, o plural de `person` é `people`. |

## Gabarito do teste

| Questão | Resposta | Explicação |

| 1 | a | `mouse` é singular e começa com som de consoante. |

| 4 | boxes | `box` recebe `-es` no plural. |
| 5 | people | `people` é o plural irregular de `person`. |
| 6 | The screen is ready. | `the screen` indica uma tela específica. |

| 9 | The | `the tickets` fala de tickets específicos e no plural. |

| 11 | companies | `company` vira `companies` no plural. |
| 12 | I have two screens. | Com plural numerado, não usamos `a/an`. |
| 13 | The children are online. | `children` é o plural irregular de `child`. |
| 14 | This branch is new. | `branch` singular não precisa de artigo após `this`. |

## Respostas da história

| Pergunta | Resposta | Observação |
|---|---|---|
| 1 | Ana está no escritório. | A história diz: `Ana is in the office.` |
| 2 | Ela tem um laptop e um teclado. | A frase usa `a laptop` e `a keyboard`. |
| 3 | O e-mail tem um ticket. | A história diz: `The email has a ticket.` |
| 4 | Duas pessoas estão em uma reunião. | `Two people` mostra o plural irregular de `person`. |
| 5 | Sim, as branches estão prontas. | A história diz: `The branches are ready.` |
