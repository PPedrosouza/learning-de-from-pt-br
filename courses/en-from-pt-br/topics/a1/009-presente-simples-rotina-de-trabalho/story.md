# História: Presente simples: rotina de trabalho

Nível: A1

## História em inglês

I am Ana. I work with software.

Every day, I start work at nine. I open my laptop and check my tickets.

In the morning, I write code. I use Linux and Git. I don't use Windows at work.

At ten, my team has a daily meeting. My manager asks, "Do you need help?" I answer, "No, I don't."

After lunch, I test the app. My coworker Leo checks the dashboard. He doesn't deploy on Friday.

At five, we deploy a small fix. The app runs well. I close my laptop and finish work.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| I am Ana. I work with software. | Eu sou Ana. Eu trabalho com software. |
| Every day, I start work at nine. | Todos os dias, eu começo o trabalho às nove. |
| I open my laptop and check my tickets. | Eu abro meu laptop e verifico meus tickets. |
| In the morning, I write code. | De manhã, eu escrevo código. |
| I use Linux and Git. | Eu uso Linux e Git. |
| I don't use Windows at work. | Eu não uso Windows no trabalho. |
| At ten, my team has a daily meeting. | Às dez, minha equipe tem uma reunião diária. |
| My manager asks, "Do you need help?" | Meu gerente pergunta: "Você precisa de ajuda?" |
| I answer, "No, I don't." | Eu respondo: "Não." |
| After lunch, I test the app. | Depois do almoço, eu testo o aplicativo. |
| My coworker Leo checks the dashboard. | Meu colega Leo verifica o painel. |
| He doesn't deploy on Friday. | Ele não faz deploy na sexta-feira. |
| At five, we deploy a small fix. | Às cinco, nós fazemos deploy de uma pequena correção. |
| The app runs well. | O aplicativo roda bem. |
| I close my laptop and finish work. | Eu fecho meu laptop e termino o trabalho. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| every day | todos os dias | Marcador comum de rotina. |
| check my tickets | verificar meus tickets | Ação comum em times de software. |
| write code | escrever código | Rotina de desenvolvimento. |
| I don't use Windows | eu não uso Windows | Negativa com `don't`. |
| Do you need help? | você precisa de ajuda? | Pergunta com `do`. |
| checks | verifica | Forma com `s` para `he/she`. |
| doesn't deploy | não faz deploy | Depois de `doesn't`, o verbo fica básico. |
| runs well | roda bem | `runs` com `it` ou coisa singular. |

## Perguntas de compreensão

1. What does Ana do in the morning?
2. Does Ana use Windows at work?
3. What does Leo check?
4. Does Leo deploy on Friday?
5. What do Ana and her team deploy at five?
